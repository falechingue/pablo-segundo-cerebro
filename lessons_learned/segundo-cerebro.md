# Segundo cérebro

## Contexto

Referência principal: imersão interna Bruno Okamoto / Pixel AI Hub, 2026-06-11.

## Aprendizado

O segundo cérebro deve usar Git/GitHub como fonte da verdade para contexto textual. O agente não deve carregar tudo no contexto; deve usar mapas, índices, busca e arquivos categorizados.

## Aplicação no workspace

- `mapa.md` aponta a estrutura.
- `People.md`, `Projects.md`, `Pendencias.md` são índices humanos.
- `daily_notes/` guarda acontecimentos.
- `/salve` consolida.
- Gbrain faz retrieval.
- Honcho/Roncho guarda memória persistente/personalidade.

## Riscos

- Over engineering.
- Crons demais.
- Dados pessoais em excesso.
- Jogar conversa bruta no GitHub em vez de sínteses.
